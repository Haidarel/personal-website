# Personal Website

My personal portfolio website built with Next.js

## About

Portfolio website showcasing my work as a Fullstack Developer from Bandung.

## Getting Started

First, run the development server:
```bash
npm run dev
# or
yarn dev
# or
pnpm dev
```

Open [http://localhost:3000](http://localhost:3000) with your browser to see the result.

## Tech Stack

- Next.js 16
- TypeScript
- CSS Modules
